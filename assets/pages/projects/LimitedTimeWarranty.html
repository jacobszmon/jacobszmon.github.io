Limited Time Warranty (Ongoing)

Project Details

Project Type: Video Game
Engine / Languages Used: Unreal Engine 5, Blueprints
Version Control Used: Perforce Helix Core (p4v)
Role: Programmer, UI Programmer, Animation Integrator

  • Itch

Description

Limited Time Warranty is a hectic, physics-based puzzle game where you must complete tasks around an office.
Welcome to Perish.inc, a company that handles warranty fulfillment.
Everything in the office abides by two simple rules:

  1. Everything is breakable.
  2. Anything can be replaced, so long as it's under warranty.

Background

LTW started as our core team's end of the year project for RRC's "Game Studio 2" class. The team was assigned by our instructors, and from the moment I saw our names on screen, I knew whatever we made would be great. The whole class was made to do a brainstorming exercise where we did three rounds of tossing ideas around with random prompts.
3 randomly generated prompts later, we had a basic idea. Then we left the classroom to find a breakout room, and fleshed that idea out for nearly an hour. When we emerged, we had something magic. A physics based puzzle game where everything was breakable, everything was replaceable within a warranty period, all wrapped in a liminal office space.

Goals

The main goal I had for this game is adding juice wherever possible. Some of my favourite stuff to do in game projects revolves around doing small things that make the game so much more fun to play, and I could tell that this game would give me plenty of space to do that.
I also wanted to use this game as an excuse to learn the best practices for UI programming in UE5, which I've heard can be a real hassle when you get into the weeds of it.

A Fun Fact About This Project

Having just come back from our school's reading week, (during which Sam, Emily and I participated in the WGC's pegjam 2026 in different groups), I had one thing on my mind. And that thing was Super Goblin Putt Putt, a game made for that jam that absolutely floored me.
It heavily inspired a lot of the ideas I pitched in the initial rounds of brainstorming.
